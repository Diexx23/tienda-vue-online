# 🛒 Tienda Vue Online

Proyecto desarrollado en Vue.js que simula una tienda virtual con carrito de compras dinámico.

## 🚀 Características

- Listado dinámico de productos
- Vista detallada de cada producto
- Carrito de compras estilizado
- Selección de cantidad por producto
- Actualización automática del total

## 🛠️ Tecnologías utilizadas

- Vue 3
- Vue Router
- JavaScript
- CSS

## 📦 Instalación

Clonar el repositorio:

```bash
git clone https://github.com/Diexx23/tienda-vue-online.git
```

Entrar a la carpeta:

```bash
cd tienda-vue-online
```

Instalar dependencias:

```bash
npm install
```

Ejecutar el proyecto:

```bash
npm run dev
```
